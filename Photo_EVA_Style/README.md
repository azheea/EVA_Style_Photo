# 开始

## 准备
 1. 你所需要修改的图片
 2. 安装python3.8+

## 开始

将图片放入.**/photo**文件夹内
使用`pip install pillow`命令安装PIL库
运行 main.py
图片就会生成在 **./processed_photos**文件夹下
